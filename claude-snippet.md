## UI Reference Assets (uisnap)

When asked to generate SVG illustrations, Lottie animations, or animated SVGs from screenshots or recordings, look in `.uisnap/` for processed reference materials. Each subdirectory contains a `manifest.json` describing the source material, extracted frames (if video/GIF), and any style or format hints provided by the user. Read the manifest first to understand what you're working with, then read the image files to understand the visual content before generating assets.

Generated assets should be written back alongside the source material or to the project's assets directory as appropriate.
